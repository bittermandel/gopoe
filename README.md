# Library for Path of Exile API 

## Features
* Premade data structures for all relevant objects (Characters, Items, Stashtab, etc)
* Utility functions (Last accessed character and more)

## Usage
```go
package main

import gopoe "github.com/bittermandel/gopoe/pkg"

func main() {
    item := gopoe.Item{}
    item.Name
}
```
